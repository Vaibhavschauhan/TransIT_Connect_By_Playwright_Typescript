# 🚀 Playwright TypeScript Automation Framework

This project is an automation testing framework built using Playwright and TypeScript. It follows Page Object Model (POM) structure for better maintainability.

---

## 📁 Project Structure

- Pages/ → Page classes (homePage.ts, contactPage.ts)
- tests/ → Test files
- TestData/ → Test data (JSON)
- playwright.config.ts → Configuration file

---

## ⚙️ Installation

1. Clone the repository:
git clone https://github.com/your-username/your-repo.git

2. Install dependencies:
npm install

3. Install Playwright browsers:
npx playwright install

---

## ▶️ Run Tests

Run all tests:
npx playwright test

Run in headed mode:
npx playwright test --headed

---

## 📊 View Report

npx playwright show-report

---

## 🧪 Features

- Cross-browser testing
- Page Object Model (POM)
- TypeScript support
- JSON test data handling
